# Term Deposit Subscription Prediction

## 📊 Overview
This project predicts whether a customer will subscribe to a term deposit using machine learning techniques.

## 🛠 Tools
- Python
- Pandas
- Scikit-learn

## 🔍 Process
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection
- Model building (Logistic Regression, Decision Tree, Random Forest)

## 📈 Results
- Random Forest achieved the best performance among tested models
- Improved prediction accuracy compared to baseline models

## 💡 Insights
- Contact duration strongly influences customer subscription decisions
- Previous campaign outcomes significantly impact prediction results
